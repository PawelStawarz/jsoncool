Detailed Description

Namespace containing presets of the jsonCool::jsonData class.

Author
mgr inz. Pawel Stawarz
Date
07 november 2015

The presets namespace contains instances of the jsonCool::jsonData that may be used to make the code look more like the JavaScript code.

Example: 

// Creates an array with 3 nulls:
jsonCool::jsonData someData;
someData = jsonCool::presets::array;
for(unsigned int i=0; i<3; i++)
    someData() = jsonCool::presets::null;
std::cout << someData << std::endl;
