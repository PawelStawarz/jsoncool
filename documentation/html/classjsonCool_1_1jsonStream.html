Detailed Description

JSON input/output stream.

Author
mgr inz. Pawel Stawarz
Date
07 november 2015

This class provides a simplified interface to interact with JSON files. A file can be opened by calling the jsonStream::open() function, or using the constructor. Data can then be accessed in various ways - either as a std::string or as an instance of jsonCool::jsonData .

Example: 

try{
    jsonCool::jsonStream input("input.json");
    if(input.data().isObject())
        input.data()["someNewProperty"] = presets::null;
    else
        input.data()() = presets::null;
    input.saveAs("inputModified.json");
}
catch(jsonException &e){
    std::cout << "Exception thrown at \'" << e.source() << "': " << e.what() << std::endl;
}
